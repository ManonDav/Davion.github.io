Voici mon tout premier projet : le jeu Snake développé en Python avec Pygame.

Jeu du moulin

Le jeu du moulin développé en Python avec Pygame.

Dans ce jeu, on joue contre l'ordinateur que j'ai codée pour qu'il essaye de nous bloquer et pour qu'il joue un peu intelligemment. Le but est de former une suite de 3 pions vertical ou horizontal, donc quand on forme un moulin on peut piquer un pion à l'adversaire. Pour gagner, il faut réussir à piquer les pions de l'adversaire jusqu'à ce qu'il ne lui en reste plus que deux.

Le jeu Sokoban

Le jeu Sokoban développé sous Java.

Le but du jeu Sokoban est de déplacer les caisses jusqu'au endroit où elles doivent être sans rester bloquer. Ce jeu est équipé d'un bouton retour qui permet d'annuler la dernière action par exemple, si on est bloqué. On déplace le petit personnage avec les flèches directionnelles. Il y a en tout 3 niveaux différents de plus en plus dure.

Le célèbre Jeu Pacman en Bash

Le jeu Pacman sous Shell

On a codé une variante du jeu Pacman avec le langage Bash. Dans notre version, vous avez le choix de créer votre propre carte ou de jouer avec les 4 maps qu'on a crée. Le but de Pacman est de réussir à manger tous les bonbons sans se faire attraper par les fantômes. Quand Pacman mange les bonbons spéciaux, il devient invulnérable pendant un petit laps de temps, et cela lui permet de manger les fantômes et de les faire réapparaître à leur point de départ.
