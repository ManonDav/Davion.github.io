Snake Game

Voici mon tout premier projet : un jeu Snake développé en Python avec Pygame.

Dans ce jeu, on contrôle un serpent à l'aide des flèches directionnelles, et l'objectif est de manger le plus de pommes possible. Plus le serpent mange de pommes, plus il grandit. La partie est perdue si le serpent touche un mur ou son propre corps.

Jeu du moulin

Le jeu du Moulin développé en Python avec Pygame.

Dans ce jeu, on joue contre l'ordinateur, que j'ai programmé pour qu'il essaie de nous bloquer et joue de manière stratégique. L'objectif est de former une suite de 3 pions alignés verticalement ou horizontalement. Lorsqu'on forme un moulin, on peut retirer un pion à l'adversaire.

Pour gagner, il faut retirer les pions de l'adversaire jusqu'à ce qu'il ne lui en reste plus que deux.

Le jeu Sokoban

Le jeu Sokoban développé en Java.

Le but du jeu Sokoban est de déplacer les caisses jusqu'aux emplacements prévus, sans se retrouver bloqué. Ce jeu inclut un bouton "Retour" qui permet d'annuler la dernière action, utile notamment en cas d'erreur. Le joueur contrôle le personnage à l'aide des flèches directionnelles.

Le jeu propose trois niveaux de difficulté croissante.

Voir le projet sur GitHub

Le célèbre Jeu Pacman en Bash

Le jeu Pacman développé en Shell.

Nous avons créé une variante du jeu Pacman en utilisant le langage Bash. Dans notre version, vous pouvez choisir de concevoir votre propre carte ou de jouer sur l'une des quatre cartes prédéfinies que nous avons créées. Le but est de manger tous les bonbons sans se faire attraper par les fantômes.

Lorsque Pacman mange un bonbon spécial, il devient invincible pendant un court laps de temps. Cela lui permet de manger les fantômes, qui réapparaissent ensuite à leur point de départ.
